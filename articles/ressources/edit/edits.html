Voici la liste des éditeurs de (appelés IDE):
Visual studio code: éditeur réputé, mais consomme beaucoup de ressources !
VSCodium: Une version de visual studio code, sans toute la télémétrie et "l'espionnage" de Microsoft; plus rapide
Notepad ++:Extrêmement recommendé pour une utilisation rapide ! Notepad ++ est un editeur de code très rapide, avec une verison portable sans installation pour l'emporter partout ! Parfait pour les développeurs du dimanche !
VSCode Web: l'édition web de visual studio code pas recommendé car consomme beaucoup de ressources, presque plus que la version non web


Avec un de ces IDEs, vous pouvez avoir une bonne session de code, rapide ou longue !
